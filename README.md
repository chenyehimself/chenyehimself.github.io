# Chenye Wang — Personal Website

This is the source code for my personal website: chenyewang.com

The site presents my work across music, creative coding, audio technology, and AI-assisted music systems. It is designed as a personal brand website rather than a traditional resume page, highlighting both my artistic identity and my technical direction.

## About

I am a Computer Music student at Johns Hopkins University’s Peabody Institute, working across music production, creative coding, audio technology, and AI-assisted music systems.

My work focuses on how technology can expand musical authorship rather than replace it. I am interested in tools that help people express intent, shape sound, and stay in control of the creative process.

## Tech Stack

This website is built as a lightweight static site and deployed through GitHub Pages.

Current structure:

txt . ├── index.html ├── style.css ├── CNAME └── README.md 

## Deployment

The site is deployed with GitHub Pages and connected to the custom domain:

txt chenyewang.com 

The CNAME file should remain in the repository root and contain:

txt chenyewang.com 

Do not remove the CNAME file unless the custom domain setup changes.

## Design Direction

The visual direction of the site is a dark, minimal, music-tech interface.

Design keywords:

- dark / near-black background
- off-white typography
- subtle violet and cyan accents
- large typography
- glass-like project cards
- thin borders
- generous spacing
- lightweight interactions
- responsive layout

The goal is to make the site feel like a mix of a music artist website, a creative technologist portfolio, and an early founder personal brand.
